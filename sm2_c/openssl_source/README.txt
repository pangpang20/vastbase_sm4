请将下载的 OpenSSL 源码包放置在此目录

需要的文件：
  openssl-3.0.18.tar.gz

下载地址（选择任一）：

1. 清华大学 TUNA 镜像（推荐，国内速度最快）
   https://mirrors.tuna.tsinghua.edu.cn/openssl/source/openssl-3.0.18.tar.gz

2. 中科大 USTC 镜像
   https://mirrors.ustc.edu.cn/openssl/source/openssl-3.0.18.tar.gz

3. 阿里云镜像
   https://mirrors.aliyun.com/openssl/source/openssl-3.0.18.tar.gz

4. OpenSSL 官方（国外，速度较慢）
   https://www.openssl.org/source/openssl-3.0.18.tar.gz

下载完成后：
- 将文件放置在本目录 (openssl_source/)
- 返回上级目录执行: sudo bash install_openssl3.sh
